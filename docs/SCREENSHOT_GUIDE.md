## üõ†Ô∏è How to Create Screenshots

### **Option 1: iPhone Screenshots (Recommended)**
1. Install app from TestFlight
2. Navigate to each screen
3. Press **Volume Up + Side Button** to screenshot
4. AirDrop to Mac
5. Resize if needed (should be correct size already)

### **Option 2: Xcode Simulator**
1. Open Xcode
2. Run app in simulator (iPhone 15 Pro Max)
3. Navigate to screen
4. **Device > Screenshot**
5. Saves to Desktop at correct size

### **Option 3: Design Tool (Figma/Sketch)**
1. Create artboard at 1290 x 2796
2. Add app screenshots
3. Add captions/annotations if desired
4. Export as PNG

## üöÄ Upload to App Store Connect

1. Log in to https://appstoreconnect.apple.com
2. Select your app (Aperioesca)
3. Go to **App Store** tab
4. Scroll to **App Previews and Screenshots**
5. Select **6.7" Display**
6. Drag and drop screenshots (in order)
7. Add captions (optional)
8. Save

## üí° Pro Tips

- **Order matters:** First screenshot is the most important (shows in search results)
- **Use real data:** Actual meal photos look better than stock images
- **Show variety:** Different meals, energy bands, confidence levels
- **Highlight uniqueness:** Crossed-out kcal logo, confidence levels, patterns
- **Keep it simple:** Don't overcrowd with text or annotations
